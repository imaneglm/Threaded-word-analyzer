# Threaded-word-analyzer
The threaded word analyzer is the threaded version of the Java program searching for words in the paragraph, word count, and line it can be found. This program displays applications of threads in multitasking.

# Features
Counts the total occurrence of a given word in the paragraph.
Finds the lines on which the word appears.
Supports multithreading for processing simultaneously:
Thread1 computes the overall frequency of the word. 
Thread2 finds all the lines which the word has been mentioned in.

# Technologies Used
Java
Java Threads
